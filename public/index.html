프롬프트 편집

분석팀은 3단계로 동작합니다: 분석가가 claim을 추출 → 검증가가 원문 근거 확인 → 작가가 한국어 리포트 작성.

분석가 — claim과 재현가능성 정보 추출 (JSON)
검증가 — claim별 원문 근거 검색 (JSON)
작가 — 한국어 리포트 작성 (Markdown)

각 에이전트별로 백엔드(Claude / Codex)를 지정합니다. Claude는 모델을 고르고, Codex는 GPT-5.5 고정 상태에서 추론 강도만 고릅니다.
채팅: 분석 후 사용자가 같은 논문에 대해 질문할 때 사용.

로그인 필요

Claude Code와 OpenAI Codex 둘 다 로그아웃 상태입니다. 둘 중 최소 하나에 로그인해야 앱을 사용할 수 있습니다.

Claude Code 로그인

  1. 터미널을 열고 claude 입력 후 Enter
  2. 안내에 따라 브라우저에서 로그인
  3. 아래 "상태 다시 확인" 클릭

OpenAI Codex 로그인

  1. 터미널에서 codex login 입력 후 Enter
  2. 안내에 따라 인증 완료
  3. 아래 "상태 다시 확인" 클릭
