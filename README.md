# Korean Paper Agent Console · v0.1.0

영어 논문 PDF 한 편을 입력하면 여러 전문 에이전트가 분석하고 각 주장(claim)에 원문 근거(evidence)를 매핑해 한국어 연구 리포트를 생성하는 데스크탑 콘솔입니다. Claude.ai 구독으로 동작하는 Claude Code CLI를 백엔드 LLM 호출에 사용합니다. v0.1은 단일 논문 분석 + claim 검증 UI까지를 범위로 합니다.

## 비용 & 프라이버시

본 도구는 사용자의 Claude.ai Pro/Max 구독으로 `claude` CLI를 subprocess 호출합니다. Anthropic API 키를 사용하지 않으며 토큰당 과금이 발생하지 않습니다. 분석한 PDF는 로컬(`data/`)에만 저장되고 외부로 업로드되지 않습니다.
